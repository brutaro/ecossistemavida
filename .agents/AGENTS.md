# Regras e Conhecimentos do Projeto Ecossistema Vida

- **Logo e Identidade Visual**: Ao trabalhar com a logo (`/public/logo-tree-transparent-pillow.png`), **sempre** mantenha o texto "ECOSSISTEMA VIDA" ao lado da imagem (nos componentes como Navbar e Footer). O texto não deve ser removido ou substituído inteiramente pela imagem.
- **Tratamento de Imagens**: Algumas imagens fornecidas podem conter um falso fundo transparente (padrão quadriculado branco e cinza embutido). Se isso ocorrer, foi utilizado um script Python (`Pillow`) para filtrar os pixels claros e convertê-los em fundo verdadeiramente transparente, resultando no arquivo atual.
